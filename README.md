# Data-Structures-and-Algorithms-
This repository contains a complete, robust implementation of a Red-Black Tree in C++. A Red-Black Tree is a self-balancing binary search tree that ensures logarithmic time complexity for fundamental operations by enforcing strict coloring and structural properties. This project demonstrates advanced data structure management, including complex node recoloring and tree rotations.
Key Features Implemented:
.Core Operations: Efficient node insertion and deletion, fully supported by structural rebalancing (left and right rotations) and recoloring mechanisms to maintain Red-Black properties.
.Tree Traversals: Includes standard Depth-First Search (DFS) traversals: Inorder, Preorder, and Postorder.
.Tree Metrics: Methods to accurately calculate the overall height and diameter (longest path between any two nodes) of the tree.
.Utility Functions: Includes node searching, finding the minimum/maximum values, and identifying in-order successors and predecessors.
.Tested Output: The implementation is thoroughly tested, successfully performing insertions, calculating metrics (e.g., a height of 3 and diameter of 5 for the test dataset), and maintaining sorting order after standard deletions.
